Anne&Max Bezoekenkaart – installatie

Dit is een Progressive Web App (PWA).

Belangrijk:
Een PWA moet via een beveiligde website (https) of lokale webserver worden geopend.
Alleen het index.html-bestand openen is niet genoeg om de app te installeren.

Eenvoudige opties:
1. Upload de map naar Netlify, GitHub Pages, Cloudflare Pages of een andere https-webhost.
2. Open daarna de website op je telefoon.
3. Android/Chrome: tik op 'Installeer app' of kies 'Toevoegen aan startscherm'.
4. iPhone/Safari: Deel-knop > Zet op beginscherm.

De bezochte filialen worden lokaal op het apparaat opgeslagen.
